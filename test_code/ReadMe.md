# 评分卡建模流程
## 一、单变量分析
1. 功能:单变量分析
    1. 变量画像:数据类型,缺失值个数,个数,缺失率,均值,标准差,最小值,中位数,最大值,唯一值个数,Top1占比,Top3占比
    2. 缺失值填充:
        - 数值型变量:均值填充,中位数填充,众数填充
        - 离散型变量:众数填充
    3. 异常值处理:异常值个数,异常值占比
    4. 离散变量编码
2. 输入:数据集
3. 输出:单变量分析报告、处理后数据集

## 二、随机抽样

2. 输入:处理后数据集
3. 输出:随机抽样数据集
